🔑 Setup Firebase (Sekali Saja)

Masukkan konfigurasi Firebase Anda:

📖 Cara mendapatkan Firebase Config
  1. Buka Firebase Console
  2. Buat project baru atau pilih project existing
  3. Klik ⚙️ Settings → Project settings
  4. Scroll ke bawah, pilih "Web app" (ikon </>)
  5. Copy nilai dari firebaseConfig
  6. Di menu kiri, klik "Realtime Database"
  7. Klik "Create Database" → pilih lokasi → Start in "Test mode"
  8. Paste konfigurasi di form di atas
